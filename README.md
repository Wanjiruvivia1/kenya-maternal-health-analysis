
# Kenya Maternal & Child Health Analysis 🏥

Exploratory data analysis of maternal and child health disparities 
across Kenya's 47 counties using KDHS 2022 data.

## Objectives
- Identify county-level disparities in maternal and child health
- Analyze factors affecting antenatal care attendance
- Visualize health inequalities across regions

## Tools
Python | Pandas | Matplotlib | Seaborn | Power BI
